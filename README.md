# Startpage

A simple, customizable startpage for your browser with search functionality, bookmarks, and a clock.

## Features

- **Search Engines**: Choose from multiple search engines including Virgil (Games), PlaySeek (Games), Google, and ChatGPT.
- **Bookmarks**: Add, edit, and manage your favorite websites with keyboard shortcuts (chords).
- **Clock**: Displays current time and date, with options for 12/24 hour format.
- **Themes**: Switch between light and dark themes.
- **Keyboard Shortcuts**:
  - Press `/` to focus the search input.
  - Use chord shortcuts (e.g., "HN" for Hacker News) to quickly open bookmarks.
- **Persistence**: All settings and bookmarks are saved in localStorage.

### Adding Bookmarks

- Click the "Add" button to create a new bookmark.
- Right-click or double-click on an existing bookmark to edit it.
- Assign chord shortcuts for quick access (e.g., "HN" for Hacker News).

### Settings

- Click the "Settings" button to change theme and time format.

## Browser Compatibility

Works in modern browsers that support ES6 features.
